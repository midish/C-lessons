Строка отличается от массива символов тем что ВСЕГДА в конце имеет знак конца строки - '\0'
char Hi[] = { 'H', 'i' } - обычный массив символов.
char Hi[] = { 'H', 'i', '\0' } - строка.

Строку так же можно декларировать таким образом char Hi[] = "Hi" или  char *Hi = "Hi"
(char Hi[] = { 'H', 'i', '\0' }) == (char Hi[] = "Hi")
Когда строка объявляется через указатель (char *Hi = "Hi";) подрузамевается то, что хранящееся значение в строке Hi является 
констатой, и при попытки изменить её может получится ошибка (Например Segmentation Fault (core dumped))
Что-бы обезопасить себя от ошибок при объявлении через указатель, можно добавить пере объявление const
const char *Hi = "Hi"; - теперь строка является констатой и не может быть изменена.

const char *hello1 = "Hello1";
const char *hello2 = "Hello2";

printf("%s", hello1);
printf("%s", hello2);
Выведет Hello1
        Hello2
Если я напишу
hello2 = hello1; то hello2 будет хранить адрес hello1 и соотвественной выведет значение хранящееся в hello1
printf("%s", hello1);
printf("%s", hello2);
Выведет Hello1
        Hello1
