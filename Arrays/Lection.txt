Имя в массиве языка Си является УКАЗАТЕЛЕМ !
prices[] == *prices

Написав *prices получаю значение по НУЛЕВОМУ индексу.
prices[4] = { 228, 322, 1488 }
*prices == prices[0] в 0 индексе храниться значение 228

Если я хочу получить доступ к какому-либо элементу массива, то могу написать так
prices[4] или *(prices + 4) (идёт сдвиг на кол-во байт равное 4 * (тип)
Во втором случает я сдвигаю по ТИПУ (например char, float, int т.д.)
